# Installation

> ![NOTE]
> This project requires [Node.js](https://nodejs.org) 18.0 or higher.

1. Clone the repository

```bash
git clone https://github.com/Susmita-Rani-Ghosh/CSC4710/
cd CSC4710
```

2. Install the dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
